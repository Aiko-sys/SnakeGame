## Installation (Windows)

Open Windows Terminal.

Clone the repository and open its folder
```
git clone https://github.com/Richxz/SnakeGame.git
```
Installpygame
```
cd SnakeGame
```
```
python -m pip install -U pygame=2.5.0 --user
```
Open The game
```
cd SnakeGame
python main.py
```
