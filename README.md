## Installation (Windows)

Open Windows Terminal.

Clone the repository and open its folder
```
git clone https://github.com/Richxz/SnakeGame.git
```
```
cd SnakeGame
```
if you don't have the pygame lib, Install.
```
start install.bat
```
Open The game
```
start SnakeGame.bat
```
